Skillcrush Projects

My journey into coding started with Skillcrush's Break into Tech course

This online program provided me with flexible training to learn the fundamentals in HTML, CSS and GIT & GITHUB. In a few more weeks, I'll then be on my way to the Skillcrush Visual Design track. Below are a few of the projects I have completed throughout this undertaking. I'm finding so much enjoyment in writing up clean & organized code that further challenges my creative brain.

Jubilee Austen

HTML + CSS
My first class project, the Jubilee Austen website was created from a given design comp. I recreated the exact design using HTML and CSS skills from classes leading up to the project.

Recipe Cards

CSS
This was a bonus project in class. We were instructed to style a recipe card website to match a given design comp, adjusting fonts, borders, blocks and lists.

Vision Board

HTML + CSS
Another bonus project, this Vision Board challenge had us practicing skills in writing divs, clearing flaots, styling text and building grids.
